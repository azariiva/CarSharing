Comparing sets

Consider two sets: set1 and set2.

Select all cases when the following statement returns true.

set1.equals(set2);

Cases:

a)

Set<Integer> set1 = Set.of(1, 2, 3);
Set<Integer> set2 = Set.of(1, 2, 3);

b)

Set<Integer> set1 = Set.of(1, 2, 3);
Set<Integer> set2 = Set.of(3, 2, 1);

c)

Set<Integer> set1 = new LinkedHashSet<>();
set1.add(1);
set1.add(3);
set1.add(2);

Set<Integer> set2 = new LinkedHashSet<>();
set2.add(1);
set2.add(3);

d)

SortedSet<Integer> set1 = new TreeSet<>();
set1.add(1);
set1.add(3);

Set<Integer> set2 = Set.of(3, 1);
